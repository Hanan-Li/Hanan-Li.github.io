    IoT Nerf Gun Shooter

    What is This?

    During the summer of the pandemic, I got bored and decided to build something stupid using a nerf gun and a raspberry pi that I owned but never used. I decided it would be fun to make a remote control IoT nerf gun shooter using some pieces of plywood lying around and a servo motor. Using an Android App, you can shoot the nerf dart from anywhere with wifi connection.

    Why is this interesting?

    I thought it was pretty funny.
